# Netqon OpenWebMonitor

网页内容监视器，网页内容变更监控，网页监控，网站内容变化检测，监控网页内容变化。支持动态内容，支持选择特定区域。

Website content change monitor tool

### Build

```
cd project
npm install
yarn dist
```

### Download

[Installer](http://openwebmonitor.netqon.com/) (prebuilds for macOS and win32)
